# Table Ready

A free, browser-based blackjack strategy trainer — built to help new players get comfortable with real casino blackjack, one hand at a time.

## Why this exists

This project has been something I always wanted to create — to help new players become capable of playing casino games, particularly blackjack, at a level where they can feel confident in their decisions.

I hope this helps everyone who's been looking for something like it, and eases any trouble you've had learning to play. I'll keep fleshing this out over time, and may add more casino games down the road.

Please enjoy!
— SledgeMakesGames

## Features

- Full blackjack engine — hit, stand, double, split, and surrender
- Live basic strategy chart with a decision-by-decision mistake tracker
- Table variants — Standard, Double Down Madness, and Free Bet Blackjack
- Side bets — Perfect Pairs, 21+3, and more
- Card counting tools — running count, true count, and cards remaining, with a cut-card placement mechanic
- Drill mode for practicing specific tricky decisions
- Live win-probability odds calculated from the actual remaining shoe
- Multiplayer bots (up to 6 seats) that play correct basic strategy and realistically work through the shoe with you
- Session stats, sound effects, and multiple table themes
- Everything saves automatically, so you can pick up right where you left off
